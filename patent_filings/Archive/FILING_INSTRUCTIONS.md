## Pre-Filing Checklist

### Micro Entity Qualification (Gross Income Basis — Form PTO/SB/15A)

Before filing, confirm ALL of the following:

- [ ] Each applicant and inventor qualifies as a small entity (individual, not assigned to entity with 500+ employees)
- [ ] Neither applicant nor any inventor has been named on more than 4 previously filed U.S. nonprovisional applications (provisional applications do NOT count toward this limit)
- [ ] Neither applicant nor any inventor had gross income exceeding $251,190 in the preceding calendar year (2025)
- [ ] No ownership interest has been assigned, granted, or conveyed to any entity exceeding the $251,190 gross income limit

### Required Forms (Download from USPTO.gov)

- [ ] **PTO/SB/16** — Provisional Application Cover Sheet (one per application)
- [ ] **PTO/SB/15A** — Certification of Micro Entity Status (Gross Income Basis) — must be filed with or before the first fee payment at micro entity rate

### Filing Method

File electronically via **USPTO Patent Center** (https://patentcenter.uspto.gov):
1. Create a USPTO.gov account if you don't have one
2. Select "File a new provisional application"
3. Upload the specification PDF (converted from the .md files below)
4. Upload any drawing sheets as separate PDFs
5. Complete the cover sheet (PTO/SB/16) online or upload
6. Upload the Micro Entity Certification (PTO/SB/15A)
7. Pay the $65 filing fee per application

### PDF Conversion Notes

When converting .md to PDF for filing:
- Use US Letter size (8.5" x 11")
- Margins: 1 inch on all sides minimum (top margin at least 0.75")
- Font: Times New Roman, 12pt (or equivalent serif font)
- Line spacing: 1.5 or double-spaced preferred
- Number all pages sequentially
- Number all paragraphs sequentially within each section (as formatted in these documents)
